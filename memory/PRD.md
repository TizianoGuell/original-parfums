# Original Parfums - Demo Landing

## Problem Statement
Dejar el proyecto como una landing page de demo para presentar la marca sin
servidor, sin ecommerce y sin integraciones.

## Architecture
- Frontend only: React + TailwindCSS
- Static content: bloques editoriales y CTAs internos
- Design system: luxury theme con palette burgundy y gold

## Current Scope
- Hero principal con propuesta de valor
- Secciones de concepto, coleccion demo y CTA final
- Navegacion por anclas dentro de la misma pagina
- Build estatico listo para deploy

## Current Status
- Capa de servidor removida del proyecto
- Front desacoplado de cualquier API
- Home reemplazada por una landing demo unica
